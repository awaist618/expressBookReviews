# Final Project: Secure Book Review API

This folder contains the Express.js implementation for the Book Review API, including authentication, book retrieval routes, and review management endpoints.

## Run Locally
```bash
npm install
node index.js
```

## Example Endpoints
- GET /books
- GET /isbn/1
- GET /author/Chinua%20Achebe
- GET /title/Fairy%20tales
- POST /register
- POST /login
- PUT /customer/auth/review/1
- DELETE /customer/auth/review/1
